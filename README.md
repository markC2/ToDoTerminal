# todo

a terminal todo list 

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/MarkC2/ToDoTerminal/main/install.sh | bash
```

## Usage:

the todo keyword alone followed by a task item is the quick jot down an idea method eg todo finish homework

on top of this other keywords available :

-  `add` opens a multi item input interface , exit with ctrl + d, each new line is a task item 

- ` done ` followed by a task item number eg 1 clears that item from list 

- `todo` todo on its own or with 'today' or 'list' displays the current todo list 

- `startup` adds instance of todo to profile to add todo list display on terminal startup
